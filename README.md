# Game Narrativo em Java

Projeto desenvolvido em Java utilizando Programação Orientada a Objetos (POO), simulando um jogo interativo baseado em narrativa.

## Funcionalidades

* Criação de personagem (Guerreiro)
* Sistema de vidas
* Interação com o Oráculo
* Estrutura com itens e bolsa
* Níveis (levels) em desenvolvimento

## Tecnologias

* Java
* VS Code
